
What this is:
-------------
These are the megahal training, brain, and dictionary files for use by crunchy.  These files provide the markov interaction.  For more information on megahal, check out the megahal site (http://megahal.alioth.debian.org/) and subbot.org (http://subbot.org/). 


How to use it:
--------------
You can install megahal on debian-based systems by doing "sudo apt-get install megahal".
